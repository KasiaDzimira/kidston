Global Text Resize Using Type Selectors

This example demonstrates how to apply text size change onclick to a large number of page elements using type or element selectors. Various elements in this page have classes or ID's attached and style sheets specifying their font size in pixels. Clicking the buttons causes the style sheet specifications to be overridden and allows a resize of these page elements.
